# Hot Shot Bakery 🔥

A fully responsive multi-page bakery website built with HTML, CSS, JavaScript and Bootstrap 5.

## Pages
- Home - Landing page with hero section and why choose us
- About - Our story, values and team
- Menu - Breads, pastries and cakes with prices in KSh
- Gallery - Photo gallery with category filter 
- Blog - Bakery news, tips and updates
- Testimonials - Customer reviews
- FAQ - Common questions with Bootstrap accordion
- Contact - Contact form with JavaScript validation

## Technologies Used
- HTML5
- CSS
- JavaScript 
- Bootstrap 5

## JavaScript Features
1. Contact form validation - checks all fields before submitting
2. Gallery filter - filter images by Breads, Pastries or Cakes

## How to Run
Open index.html in any browser.
